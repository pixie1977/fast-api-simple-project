
# 🩺 Diabetes Prediction API

Сервис для предсказания риска диабета у пациента на основе антропометрических и биохимических данных.  
Реализован на FastAPI, использует ONNX-модель для инференса и защищён Basic Auth.

---

## 🔧 Технологии

- **FastAPI** — для создания JSON API
- **ONNX Runtime** — выполнение модели машинного обучения
- **Basic Auth** — защита эндпоинтов
- **Pydantic** — валидация входных данных
- **Docker** — контейнеризация
- **Логирование** — отслеживание запросов

---

## 🚀 Запуск

### Локально

bash

-> Установите зависимости
pip install -r requirements.txt

-> Запустите сервер
uvicorn src.main:app --host 127.0.0.1 --port 8000 --reload

Откройте: [http://127.0.0.1:8000](http://127.0.0.1:8000)

---

### Через Docker

bash

-> Сборка образа
docker build -t diabetes-api .

-> Запуск контейнера
docker run -p 8000:80 diabetes-api

Откройте: [http://localhost:8000](http://localhost:8000)

## 🔐 Аутентификация

Для доступа к `/predict` требуется **HTTP Basic Authentication**:

- **Username**: `demo_user`
- **Password**: `demo_pass`

Пример заголовка:
Authorization: Basic ZGVtb191c2VyOmRlbW9fcGFzcw==

## 📥 Эндпоинт: `POST /predict`

Принимает JSON с данными пациента и возвращает бинарное предсказание.

### Входные данные (JSON)

| Поле         | Тип    | Диапазон     | Описание |
|--------------|--------|-------------|---------|
| `Pregnancies`| `int`  | 0–20        | Количество беременностей |
| `Glucose`    | `float`| 50–200      | Уровень глюкозы (mg/dL) |
| `BMI`        | `float`| 15–50       | Индекс массы тела |
| `Age`        | `int`  | 18–90       | Возраст |

### Пример запроса


bash curl -X POST "http://127.0.0.1:8000/predict"
-H "Content-Type: application/json"
-u demo_user:demo_pass
-d '{ "Pregnancies": 2, "Glucose": 140, "BMI": 35.5, "Age": 32 }'
### Ответ (успешный)

json { "prediction": 1 }
> `1` — высокая вероятность диабета  
> `0` — низкая вероятность

### Ошибки

| Код | Причина |
|-----|--------|
| `401` | Неверные логин/пароль |
| `422` | Некорректные данные (например, `Glucose = -1`) |
| `500` | Ошибка выполнения модели |

---

## 🖥️ Веб-интерфейс

На главной странице (`/`) доступен простой интерфейс для ввода данных и получения предсказания:

- Форма с полями: `Pregnancies`, `Glucose`, `BMI`, `Age`
- Поля валидируются по диапазонам
- Пароль вводится скрыто (`type="password"`)
- Результат отображается в формате JSON

---

## 📊 Модель

- Формат: **ONNX**
- Путь: `model/diabetes_model.onnx`
- Вход: `[Pregnancies, Glucose, BMI, Age]` (1x4)
- Выход: вероятность диабета → порог 0.5
